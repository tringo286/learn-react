# Lesson 4: Props

This lesson will cover:

- Props
- PropTypes
- DefaultProps and Default Parameters

---

## Project Structure

- **App.jsx**: The main entry point of the application.
- **components/**: Contains the following components:
  - **Student.jsx**: A reusable component that demonstrates the use of props.

### 1. Props

**Props** (short for "properties") are used pass data from one component to another, typically from a parent component to a child component.

### 2. PropTypes

**PropTypes** is a built-in feature of React that provides a way to validate the types of props passed to components

### 3. DefaultProps and Default Parameters

**DefaultProps** is a built-in feature of React that are used to define default values for props in a component.

**Defaul Parameters** are used to set default values directly when destructuring props
